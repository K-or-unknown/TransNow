## About

The program is a translator with a graphical interface and some unique features. These include options to hide the window (running as a process in the system tray) and translate selected text using a predefined hotkey. Currently, only two languages ​​are supported, but more will be added in the future.

## Installation

To install the program, you need to follow the link below and complete the installation process.

## How to use?

**1.** Select the languages ​​you want to translate into.

**2.** Click on the text field and then press a key on the keyboard.

**3.** Click on the button to bind the key.

***Once the key is bound, it cannot be used for any other tasks except translating the selected text and later copying the translation!***

**4.** If you need to, you can click on the button to hide the window (it will be in the system tray; if you want to open the window from the system tray, just left-click and the program will open)

***To exit the program, find the program icon in the system tray, right-click on it and click "Exit"!***
